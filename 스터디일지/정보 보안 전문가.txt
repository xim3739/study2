*기업의 정보 유출을 막고 바이러스 침투 방지, 탐지, 분석 업무
*시스템 보안 해킹과 네트워크 보안 해킹을 다를줄 알아야 함
-> 해킹을 하려면 네트워크, 프로그래밍 언어, 서버 3가지 분야를 알아야한다.
*프로그래밍 기술까지 있으면 보안 백신프로그램 개발 분야로 진출 가능

-모의 해킹 전문가 : 임의로 기업을 해킹하여 취약점을 파악하고 분석하는 직업으로서, 실제로 해커가 사용하는 해킹 툴, 기법
		등으로 해커들이 어느경로로 침투할 가능성을 진단
-침해 대응 전문가 : 해킹이 들어왔을때 피해 규모를 최소화하기 위하여 사고를 보호함. 그리고 시스템을 복구하는 예방전략을
		세우는 역할
-보안 솔루션 전문가 : 백신 프로그램을 개발하여 예방하는 연구
-보안 관제 전문가 : 24시간 해킹여부를 감시하고 대응하며, 해커들이 행각지 못한 루트로 침범하지 않게 해커 침입을 분석 함

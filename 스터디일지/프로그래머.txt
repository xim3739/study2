- 컴퓨터의 언어를 이용하여 프로그램의 설계, 코딩, 디버깅 및 테스트 과정을 통하여
   컴퓨터 프로그램을 개발
- 게임회사, 웹개발사, 모바일 회사, 보안회사, 프로그램 제조사 등
- 컴퓨터 프로그램언어를 사용해서 프로그램을 코딩
- 코딩된 프로그램에 대해서 논리적인 에러를 추적 발견하고 이를 수정
- 프로그래머들은 가능한 경우, 프로그램 코드가 구조적 프로그래밍 방식에 따라 작성
- 프로그램 테스팅은 단위별, 모듈별 및 통합 테스팅의 단계별로 실시하여 완벽을 기함
- 통합된 프로그램이 그 요구 사항들을 충족하는가를 판단하기 위해 기능 테스트 실시
- 해당 프로그램이 요구 명세와 모든 일반적인 성능 요구 조건을 충족하는가를 판단하기 위해
  수용 테스트 실시
- 실제 운영 목정을 달성하기 위한 테스트 데이터 또는 테스트 시나리오를 작성
- 정보 기술의 구조를 파악하고 시스템의 설계 원칙과 사양에 근거하여 정보 시스템이 구축되었는지를 평가
- 테스트의 실제적인 결과는 예측된 결과와 비교 분석하여 에러를 수정
- 코딩 및 테스팅 과정의 모든 단계를 문서화
- 정보 기술 및 비즈니스 시스템이 적절히 통제, 모니터, 평가 되고 있는지를 확인하고, 안정된 감리 기준과 치침에
 정보시스템을 감리

알고리즘 개발자 : 알고리즘을 개발하는 사람. 컴공 분야의 알고리즘 개발자가 새로운 알고리즘을 발표하면
		타 분야의 알고리즘 개발자들이 자기 분야로 가지고 가서 적용한다.

보안 개발자 : 해킹 및 정보보안 관련 기술들을 다루는 프로그래머를 의미 함. 이를 위해 많은 기반 지식이 필요함.
	      서버 등의 플랫폼의 보안성 향상을 위해 툴을 준비하거나 서버 프로그래밍 자체를 튜닝하고 감시할
	     체제를 준비해야 함. 운영 체제, 네트워크에 대해 매우 심도깊은 이해가 필요하고 C언어를 매우 잘 다룰 줄
	    알아야 한다.

보안 오퍼레이터 : 국내에선 보안 관리자, 보안 시스템 유지보수 전문가 로 부른다. 국내엔 오퍼레이터 전문 육성 코스가
		없어서 프로그래밍을 하다가 넘어오는데 기초부터 새로 배워야 하는 경우가 많다. 프로그래머의 하위
		분류가 아니라 이웃 분류이다. 개발자와 달리, 운영 중인 시스템을 지속적으로 모니터링 하다가 보안침해
		의심 사고 발생시 신속하게 조치해서 차단하고 이미 입은 피래를 파악해 복구하는 일을 한다.

소프트웨어 아키텍트(SA) : 전략, 조직 역학, 프로세스, 커뮤니케이션, 리더십 등 관리 능력과 엔지니어링에 대한 깊은 이해를
			갖춘 개발 지휘자. 프로그래밍과 기획 능력 둘 다 필요하다.

게임 개발자 : 프로그래밍을 통해 맵 디자인, 캐릭터 디자인, 사운드, 각종 시스템 등을 뒤섞어, 게임이라는 하나의 결과물을
	     만드는 직업. (게임 플레이 프로그래머, 물리 엔진, 그래픽/랜더링, 개발 도구, 네트워크 프로그래머)
	     최소한 한 개 이상의 프로그래밍 언어를 알아야한다. 게임사 마다 사용하는 프로그래밍 언어가 다르다.
	     그래서 단 하나의 언어만 알고있어서는 게임을 만들기 어렵다.

웹 개발자 : 웹 개발은 웹 상에서 사용되는 소프트웨어 개발과 웹페이지를 제작하는 분야. 이를 제작하는 개발자 이다.
	프론트 엔드 : 사용자가 보는 화면을 개발, 주로 HTML, CSS, JavaScript를 활용하여 웹 디자인을 웹 페이지로
		     구현하는 개발자
	백 엔드 : 웹 상에서 사용되는 소프트웨어나 웹 페이지가 기능적으로 동작하도록 구현하는 개발자. 여기서 기능이란
		해당 소프트웨어가 가진 목적 구현을 위한 모든 환경을 의미한다. 즉, 클라이언트의 기능 뿐만 아니라
		서버 개발도 해당 된다. Java, JSP, PHP, Spring Framework, 디자인 패턴 지식등 다양한 기술이 요구 됨
	풀 스택 : 프론트엔드와 백엔드를 모두 다룰 수 있는 개발자

모바일 개발자 : 스마트폰 애플리 케이션을 개발하는 사람. 안드로이드, iOS 두 가지 모바일 운영체제의 소프트웨어를 개발.
	안드로이드 APP 개발 : 안드로이드는 Java를 기반으로 두고 있는 언어이기 때문에 java에 대한 지식이 필요하며
			    안드로이드 스튜디어 3.0 부터 표준 언어로 채택하고 있는 Kotlin 또한 알아야 한다.
	iOS APP 개발 : iOS는 macOS의 언어와 동일하게 Objective-C와 Swift에 대한 지식이 필요하다.
	하이브리드 APP 개발 : HTML5로 발전, CSS3의 발전 등을 통해 제약이 많던 이전 웹 언어들과 달리 현재는 다양한
			    애니메이션과 기능이 추가되었으므로 이를 활용해 OS의 제약 없이 개발할 수 있게 되었다.
